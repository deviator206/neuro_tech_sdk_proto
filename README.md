Step To Configure and Run App.
-------------------------------


1. Clone or download application from feature/basic branch.
2. Import the downloaded application to your favourite IDE as "Maven Project".
3. Import the dependencies from lib folder of application to the classpath.
4. Please ensure you have surveillance sdk installed.
5. Create a ODBC MySQL Connecter using systems ODBC data source app.
   Mention a datasource name, mysql server name and its user and password.
6. Build and Run the application
